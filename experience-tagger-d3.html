Extensible Navigation Pattern

This page demonstrates a standard, extensible navigation menu built with D3.js using linked nodes. The implementation is inspired by the career-paths.html navigation network.

Key Features

• Force-directed layout: Nodes automatically position themselves using physics simulation
• Interactive nodes: Click to select, hover for visual feedback
• Color coding: Three tiers (primary, secondary, tertiary) for visual hierarchy
• Linked connections: Lines show relationships between navigation items
• Mobile responsive: Adapts to different screen sizes
• Extensible: Add new nodes by updating the navigationData object

How to Extend

To add new navigation nodes, modify the navigationData object in the JavaScript section:

1. Add nodes to the nodes array with properties: id, label, tier, url
2. Define relationships in the links array with source and target
3. The D3 force simulation will automatically layout the network
